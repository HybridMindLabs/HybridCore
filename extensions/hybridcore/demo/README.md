# Demo

A reference HybridCore extension showing the basics: permissions, an admin
navigation item, a dashboard widget, an admin Inertia page, a web route,
a settings link and en/bg translations.

## Install

1. **Admin → Extensions → Sync from disk**
2. Enable **Demo**
3. Its migrations (if any) run automatically on enable.
